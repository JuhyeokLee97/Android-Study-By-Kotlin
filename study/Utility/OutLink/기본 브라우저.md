# 기본 브라우저
기본 브라우저로 링크로 연결된 웹페이지 열기

``` kotlin
private fun startDefaultBrowser(context: Context, link: Uri) {
        Intent(Intent.ACTION_VIEW).apply {
            data = link
            addFlags(Intent.FLAG_ACTIVITY_CLEAR_TASK)
            addFlags(Intent.FLAG_ACTIVITY_NEW_TASK)
        }.let(::startActivity)
    }
```
